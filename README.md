# wsl2
getestet 11/2021 mit Windows 11  
> Infos gibt es hier: https://github.com/microsoft/wslg/  
* starte Windows PowerShell und aktualisiere WSL mit:   
`wsl --update`  
* lass dir die verfügbaren Linus Distributionen anzeigen:  
`wsl --list --online`  
(Stand 17.11.2021)   
